    HTML - HyperText Markup Language

    HTML, HyperText Markup Language, traduzido para Linguagem de Marcação de Hipertexto é a linguagem de marcação usada para criar e projetar páginas web. Serve para definir a estrutura e o layout das páginas webs, ao qual incorporam textos, imagens, links, formulários, tabelas, videos, iframes entre outros elementos. O HTML é uma das ferramentas primordiais do WWW (World Wide Web), ao qual é comumente desenvolvido juntamente a CSS e JS.

    Criado na década de 1990 por Tim Berners-Lee como parte do projeto para criar a WWW. Seu objetivo era facilitar o compartilhamento de documentos e informações entre pesquisadores da CERN, empresa ao qual ele trabalhava, porém o projeto se expandiu com proporções inemaginaveis.

    Atualmente, o HTML se encontra na versão HTML5, que trouxe melhorias significativas em relação ao HTML4. As principais mudanças que podem ser destacada são as melhorias de semântica, com adições de tags como header footer section nav e asive. Outras importantes adições foram as validações de formulários, elementos multimídias, a tag canvas, entre outros.

    Tags básicas em HTML

    • <html> - É a raíz do documento, serve como container de todas as outras tags em um documento HTML;
    • <head> - Onde ficam as informações e metadados do documento, como imports e o título e favicon da página;
    • <body> - Engloba todo o conteudo visível da página, é o nó pai de todo documento HTML;
    • <header> - Representa a seção de cabeçalho da página;
    • <main> - Representa a seção principal da página;
    • <div> - Container genérico, sem valor semântica e normalmente utilizada para definir layouts;
    • <footer> - Representa a seção de rodapé da página;
    • <hr> - Representa uma divisória, normalmente marcada como uma linha horizontal;
    • <h1 até h6> - São os títulos da página, em ordem semântica decrescente de importância;
    • <p> - Define um parágrafo de texto;
    • <ul> - É o container de uma lista não ordenada;
    • <li> - Representa cada elemento de uma lista;
    • <br> - Insere uma quebra de linha no texto.

    Outras tags em HTML

    • <link> - Utilizado no head do documento para importar arquivos externos;
    • <section> - Representa uma separação semântica no documento;
    • <nav> - É uma maneira semântica de criar menus de navegação;
    • <canvas> - Fornece um espaço de criação de gráficos atravéz de JavaScript;
    • <form> - Cria um formulário HTML;
    • <ol> - É o container de uma lista ordenada;
    • <a> - Direciona o elemento clicado até a pagina desejada atravez do atributo href;
    • <table> - Cria u,a tabela HTML;
    • <input> - Dependendo do type do input, cria um elemento que pega entradas do usuario;
    • <label> - Redireciona o focus do usuario para o input correspondente quando clicado;
    • <button> - Botão;
    • <style> - Inicia ou linka CSS no documento;
    • <figure> - Elemento semântico para imagens com legenda;
    • <strong> - Deixa o texto em negrito de maneira semântica;
    • <em> - Deixa o texto em italico de maneira semântica;
    • <span> - Cria uma container inline semelhante a div;
    • <img> - Adiciona uma imagem através do atributo src;
    • <iframe> - Roda outra página dentro da página atual, muito utilizado com Google Maps ou Youtube.
